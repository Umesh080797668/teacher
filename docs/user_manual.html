11. Troubleshooting

Common Issues and Solutions

üî¥ Cannot Login

Possible Causes:

  • Incorrect email or password
  • Email not verified
  • Account suspended
  • Internet connection issues

Solutions:

  1. Double-check your email and password
  2. Use "Forgot Password" to reset credentials
  3. Check your email for verification link
  4. Verify internet connection
  5. Clear app cache and try again
  6. Contact support if issue persists

üî¥ App Crashes or Freezes

Solutions:

  1. Force close the app and reopen
  2. Restart your device
  3. Check for app updates in store
  4. Clear app cache: Settings ‚Üí Apps ‚Üí Eduverse ‚Üí Clear Cache
  5. Ensure sufficient storage space (at least 100MB free)
  6. Reinstall the app if problems continue

üî¥ Data Not Syncing

Solutions:

  1. Check internet connection
  2. Pull down to manually refresh
  3. Go to Settings ‚Üí Data & Storage ‚Üí Force Sync
  4. Check if you're in offline mode
  5. Ensure auto-sync is enabled
  6. Log out and log back in

üî¥ Notifications Not Working

Solutions:

  1. Check device notification settings
  2. Enable notifications in app settings
  3. Ensure battery optimization is off for this app
  4. Check Do Not Disturb mode
  5. Reinstall app to reset notification permissions

üî¥ Cannot Generate Reports

Solutions:

  1. Check if you have data for the selected date range
  2. Ensure stable internet connection
  3. Try a smaller date range
  4. Check subscription plan (some reports need premium)
  5. Clear app cache and retry

üî¥ QR Code Not Scanning

Solutions:

  1. Clean your camera lens
  2. Ensure good lighting
  3. Hold device steady
  4. Check camera permissions
  5. Generate new QR code (old one may have expired)

Performance Issues

Issue Solution
Slow loading Clear cache, check internet speed, reduce image quality in settings
High battery drain Disable background sync, reduce notification frequency, use dark mode
Storage full Delete old backups, clear cache, export and archive old data
Data usage high Enable Wi-Fi only sync, reduce image quality, limit background data

Error Messages

  • "Session Expired": Log out and log back in
  • "Server Error": Wait a few minutes and retry, or contact support
  • "Invalid Data": Check input format, ensure required fields are filled
  • "Permission Denied": Enable required permissions in device settings
  • "Network Error": Check internet connection and retry
